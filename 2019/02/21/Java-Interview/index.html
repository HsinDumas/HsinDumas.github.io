数据结构类

LinkedList和ArrayList

顾名思义，一个是linked一个是array[^array]

array的访问和尾部插入效率高，删除以及中部插入效率低，因为要移动数组；linked插入删除效率高，访问效率低，因为要遍历[^遍历]。

HashMap

无序

TreeMap

有序

LinkedHashMap

有序
[^array]: 动态数组，因为可以扩容所以是动态的。JDK1.8中扩容是1.5倍，最大的数组容量是Integer.MAX_VALUE-8，对于空出的8位，目前解释是 ：①存储Headerwords；②避免一些机器内存溢出，减少出错几率，所以少分配③最大还是能支持到Integer.MAX_VALUE（当Integer.MAX_VALUE-8依旧无法满足需求时）
[^遍历]: JDK中是通过index和size>>1比大小来决定在哪一半中遍历
